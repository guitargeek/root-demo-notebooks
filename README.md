# ROOT Demo Notebooks

Collection of notebooks to showcase some [ROOT](https://root.cern.ch/) capabilites that could be interesting for users outside of particle physics.

Welcome to **ROOT Demo Notebooks**! This is a small but hopefully growing collection of interactive examples showcasing the power of **ROOT** for scientistsand engineers *beyond particle physics*.

These notebooks demonstrate how you can combine ROOT features like:

- **C++ performance**
- **Python interactivity**
- **Automatic differentiation with Clad**

## To lean more about ROOT

If you want to learn more about ROOT, there are tons of resources oriented towards particle physics already.

Some good places to start:

  * The [ROOT manual](https://root.cern/manual/) to learn about different parts of ROOT
  * The [ROOT tutorials](https://root.cern/tutorials/)
  * The [2026 advanced ROOT course](https://indico.cern.ch/event/1636845/overview) if you want more demo notebooks
